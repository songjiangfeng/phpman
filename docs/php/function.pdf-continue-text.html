PDF_continue_text

(PHP 4, PECL pdflib >= 1.0.0)

PDF_continue_text — Output text in next line

说明

PDF_continue_text ( resource $p , string $text ) : bool

Prints text at the next line. 成功时返回 TRUE， 或者在失败时返回 FALSE。
