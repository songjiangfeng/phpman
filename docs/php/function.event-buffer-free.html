event_buffer_free

(PECL libevent >= 0.0.1)

event_buffer_free — Destroy buffered event

说明

event_buffer_free ( resource $bevent ) : void

Destroys the specified buffered event and frees all the resources associated.

参数

bevent

Valid buffered event resource.
