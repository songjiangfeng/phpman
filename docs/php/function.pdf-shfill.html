PDF_shfill

(PECL pdflib >= 2.0.0)

PDF_shfill — Fill area with shading

说明

PDF_shfill ( resource $pdfdoc , int $shading ) : bool

Fills an area with a shading, based on a shading object.

This function requires PDF 1.4 or above.
