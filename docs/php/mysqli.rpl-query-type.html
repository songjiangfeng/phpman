mysqli::rpl_query_type

mysqli_rpl_query_type

(PHP 5, PHP 7)

mysqli::rpl_query_type -- mysqli_rpl_query_type — 返回 RPL 查询类型

说明

面向对象风格

mysqli::rpl_query_type ( string $query ) : int

过程化风格

mysqli_rpl_query_type ( mysqli $link , string $query ) : int

返回 MYSQLI_RPL_MASTER， MYSQLI_RPL_SLAVE 或 MYSQLI_RPL_ADMIN，具体的返回值取决于查询的类型。 INSERT，UPDATE 以及类似的操作属于 主机 查询，SELECT 属于 从机查询，FLUSH 和 REPAIR 以及类似的操作属于 管理 查询。

Warning
