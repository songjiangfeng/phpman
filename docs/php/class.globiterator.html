The GlobIterator class

(PHP 5 >= 5.3.0, PHP 7)

简介

遍历一个文件系统行为类似于 glob().

public __construct ( string $pattern [, int $flags = FilesystemIterator::KEY_AS_PATHNAME | FilesystemIterator::CURRENT_AS_FILEINFO ] )

}
