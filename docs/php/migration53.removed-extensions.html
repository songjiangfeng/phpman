被移除的扩展

这些扩展已经被移动至 PECL 并且不再是 PHP 发布的一部分. 这些扩展的 PECL 包版本将根据用户需求创建.

  • dbase - 不再被保持
  • fbsql - 不再被保持
  • fdf - 被保持
  • ming - 被保持
  • msql - 不再被保持
  • ncurses - 被保存
  • sybase - 停用; 使用 sybase_ct 扩展代替.
  • mhash - 停用; 使用 hash 扩展代替. hash 全兼容 mhash; 全部使用旧函数的应用程序仍将可以工作.
