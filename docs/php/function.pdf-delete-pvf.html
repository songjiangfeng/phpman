PDF_delete_pvf

(PECL pdflib >= 2.0.0)

PDF_delete_pvf — Delete PDFlib virtual file

说明

PDF_delete_pvf ( resource $pdfdoc , string $filename ) : int

Deletes a named virtual file and frees its data structures (but not the contents).
