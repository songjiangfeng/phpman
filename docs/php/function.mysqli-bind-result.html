mysqli_bind_result

(PHP 5 < 5.4.0)

mysqli_bind_result — mysqli_stmt_bind_result() 的别名

说明

此函数是该函数的别名： 此函数是 mysqli_stmt_bind_result() 的一个别名。

Warning

本函数已自 PHP 5.3.0 起废弃并将自 PHP 5.4.0 起移除。
