hw_api_content::read

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api_content::read — Read content

说明

hw_api_content::read ( string $buffer , int $len ) : string

Reads len bytes from the content into the given buffer.

参数

buffer

len

Number of bytes to read.

返回值
