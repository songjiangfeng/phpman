gupnp_service_proxy_remove_notify

(PECL gupnp >= 0.1.0)

gupnp_service_proxy_remove_notify — Cancels the variable change notification

说明

gupnp_service_proxy_remove_notify ( resource $proxy , string $value ) : bool

Cancels the variable change notification.

参数

proxy

A service proxy identifier.

value

The variable to add notification for.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

参见

  • gupnp_service_proxy_add_notify() - Sets up callback for variable change notification
