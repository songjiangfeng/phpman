PDF_show

(PHP 4, PECL pdflib >= 1.0.0)

PDF_show — Output text at current position

说明

PDF_show ( resource $pdfdoc , string $text ) : bool

Prints text in the current font and size at the current position. 成功时返回 TRUE， 或者在失败时返回 FALSE。
