APCIterator::key

(PECL apc >= 3.1.1)

APCIterator::key — 获取迭代器的键

说明

public APCIterator::key ( void ) : string

获取当前迭代器的键。

参数

此函数没有参数。

返回值

成功时返回一个键，失败时返回 FALSE。

参见

  • APCIterator::current() - 获取当前项
  • Iterator::key() - 返回当前元素的键
