HaruPage::endText

(PECL haru >= 0.0.1)

HaruPage::endText — End current text object

说明

HaruPage::endText ( void ) : bool

Finalizes current text object.

参数

此函数没有参数。

返回值

Returns TRUE on success.

错误／异常

发生错误时抛出 HaruException 异常。
