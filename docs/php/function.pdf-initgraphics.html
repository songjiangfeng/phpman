PDF_initgraphics

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_initgraphics — Reset graphic state

说明

PDF_initgraphics ( resource $p ) : bool

Reset all color and graphics state parameters to their defaults. 成功时返回 TRUE， 或者在失败时返回 FALSE。
