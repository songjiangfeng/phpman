gupnp_service_action_return

(PECL gupnp >= 0.1.0)

gupnp_service_action_return — Return successfully

说明

gupnp_service_action_return ( resource $action ) : bool

Return successfully.

参数

action

A service action identifier.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
