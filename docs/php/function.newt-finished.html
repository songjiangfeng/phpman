newt_finished

(PECL newt >= 0.1)

newt_finished — Uninitializes newt interface

说明

newt_finished ( void ) : int

Uninitializes newt interface. This function be called, when program is ready to exit.

返回值

Returns 1 on success, 0 on failure.
