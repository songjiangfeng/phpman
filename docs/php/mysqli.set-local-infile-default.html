mysqli::set_local_infile_default

mysqli_set_local_infile_default

(PHP 5, PHP 7)

mysqli::set_local_infile_default -- mysqli_set_local_infile_default — 取消用户指定的回调函数

说明

mysqli_set_local_infile_default ( mysqli $link ) : void

使之前通过 mysqli_set_local_infile_handler() 设置的 LOAD DATA INFILE LOCAL 回调函数失效。

参数

link

仅以过程化样式：由mysqli_connect() 或 mysqli_init() 返回的链接标识。

返回值

没有返回值。

范例

参见 mysqli_set_local_infile_handler() 例程
