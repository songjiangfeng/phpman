扩展中的其他变化

扩展中行为和新特性的变化:

  • mysqli - mysqli_result 现在实现 Traversable
  • pdo_mysql - 不再支持使用低于 4.1 版本的 MySQL 客户端库连接。
  • MySQL 扩展 mysql 、mysqli 及 PDO_mysql 现在使用 mysqlnd 作为默认库。通过在配置选项中指定路径仍然可以使用 libmysqlclient 。
  • mysqlnd - 增加支持命名管道
