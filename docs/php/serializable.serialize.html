Serializable::serialize

(PHP 5 >= 5.1.0, PHP 7)

Serializable::serialize — 对象的字符串表示

说明

abstract public Serializable::serialize ( void ) : string

返回对象的字符串表示。

Note:

这个方法担当着对象析构器的角色。在此方法之后，__destruct() 方法将不会被调用。

参数

此函数没有参数。

返回值

返回对象的字符串表示或者 NULL 。

错误／异常

如果返回除了字符串或 NULL 之外的其他类型，将抛出 Exception。
