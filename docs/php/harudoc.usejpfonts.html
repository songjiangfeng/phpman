HaruDoc::useJPFonts

(PECL haru >= 0.0.1)

HaruDoc::useJPFonts — Enable builtin Japanese fonts

说明

HaruDoc::useJPFonts ( void ) : bool

Enables builtin Japanese fonts.

参数

此函数没有参数。

返回值

Returns TRUE on success.

错误／异常

发生错误时抛出 HaruException 异常。
