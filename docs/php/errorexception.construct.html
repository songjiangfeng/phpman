ErrorException::__construct

(PHP 5 >= 5.1.0, PHP 7)

ErrorException::__construct — 构造一个异常（Exception）

说明

public ErrorException::__construct ([ string $message = "" [, int $code = 0 [, int $severity = E_ERROR [, string $filename = __FILE__ [, int $lineno = __LINE__ [, Exception $previous = NULL ]]]]]] )

构造一个异常（Exception）。

参数

message

抛出的异常消息内容。

code

异常代码。

severity

异常的严重级别。

Note:

severity 可以是任意 integer 值，即错误常量里面的值。

filename

抛出异常所在的文件名。

lineno

抛出异常所在的行号。

previous

异常链中的前一个异常。

版本 说明
