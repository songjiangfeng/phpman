PDF_activate_item

(PECL pdflib >= 2.0.0)

PDF_activate_item — Activate structure element or other content item

说明

PDF_activate_item ( resource $pdfdoc , int $id ) : bool

Activates a previously created structure element or other content item. 成功时返回 TRUE， 或者在失败时返回 FALSE。
